# Carlwantbuddy Portfolio

A modern, responsive portfolio website showcasing my projects and skills in web development, database management, and full-stack development.

## Features

✨ **Modern Design** - Clean and professional interface with smooth animations

📱 **Fully Responsive** - Works seamlessly on desktop, tablet, and mobile devices

🎨 **Beautiful UI** - Gradient backgrounds, smooth transitions, and intuitive navigation

📊 **Project Showcase** - Displays all your GitHub repositories with descriptions

🚀 **Performance Optimized** - Fast loading times and smooth user experience

## Sections

- **Home** - Eye-catching hero section with call-to-action
- **About** - Personal introduction and skill cards
- **Projects** - Grid display of all GitHub repositories
- **Contact** - Links to social profiles and GitHub

## Technologies Used

- HTML5
- CSS3 (with gradients, flexbox, grid)
- JavaScript (ES6+)
- Font Awesome Icons

## Getting Started

1. Clone this repository
2. Open `index.html` in your browser
3. Customize the content as needed

## Customization

### Update Project Data
Edit the `projects` array in `script.js` to add or modify projects.

### Change Colors
Modify the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --accent-color: #ec4899;
}
```

### Update Personal Info
Edit the following in `index.html`:
- Your name in the navbar
- Bio and skills in the about section
- Contact links in the contact section

## GitHub Pages

To deploy this portfolio on GitHub Pages:

1. Go to repository Settings
2. Scroll to "Pages" section
3. Select "main" branch as source
4. Your portfolio will be live at: `https://carlwantbuddy.github.io/Aplacador-Portfolio`
